D2L Bookmarklets

Hacks and dirty little tricks to make Desire2Learn's WYSIWYG content editor do what I say, not what I do.

Download this project as a .zip file Download this project as a tar.gz file

Desire2Learn's content editor makes it easy to edit content files, especially for inexperienced users. However, like any WYSIWYG editor, the HTML it outputs is â€¦ sub-optimal. And it has some quirks and downright annoyances.

Fortunately, with a little JavaScript, we can fix some of those. This repo has bookmarklets to work some magic on Desire2Learn pages.

Bookmarklets

Display Raw LaTeX

I know, crazy idea, right? This bookmarklet replaces those ugly yellow image placeholders with the actual, honest-to-goodness raw LaTeX you typed in. To edit your LaTeX, just click on the raw string, and it will revert to the placeholder image. The bookmarklet also takes care of that bug wherein D2L's editor doesn't recognize your \( inline equation blocks \).

Drag this link to your bookmarks bar to install: D2L - Display LaTeX
